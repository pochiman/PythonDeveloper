016 : Global Electronics Retailer

Project Overview

This project involved building a comprehensive solution for Maven Electronics, a global electronics retailer experiencing revenue decline since 2020. The project involved ingesting data from multiple CSV files, building a robust relational data model using Excel's Power Query and Data Model features, and creating an interactive dashboard with Pivot Tables and Pivot Charts for management to explore performance metrics.

The solution provides actionable insights into sales trends, customer behavior, and operational efficiency, helping leadership make data-driven decisions to reverse the revenue decline.

Business Context

Maven Electronics sells computers, cell phones, TVs, cameras, and appliances through both online and physical stores worldwide. Despite a diverse product portfolio and global presence, the company has faced consistent revenue decline since 2020. Management needed a consolidated view of performance data to identify root causes and develop targeted recovery strategies.

Primary Question: How can we transform disparate CSV data files into an intuitive, interactive reporting system that helps management understand performance trends, identify problem areas, and make informed strategic decisions?

Methodology & Analysis

1 Data Profiling & Preparation

  • Ingested and profiled data from 5 CSV files: Sales, Products, Stores, Customers, and Exchange Rates
  • Conducted comprehensive data quality assessment across 10,000+ transactions
  • Engineered customer age ranges and standardized geographic data
  • Created custom calendar table with day, week, month, quarter, and year hierarchies
  • Implemented data validation and error handling procedures

2 Relational Data Modeling

  • Designed star schema data model with fact and dimension tables
  • Established 1:many relationships between Sales fact table and dimension tables
  • Created custom ExchangeKey for currency exchange rate relationships
  • Implemented product category and subcategory dimension tables (bonus objective)
  • Optimized model for performance and usability

3 Data Enrichment & Exploration

  • Created dedicated measures table for consistent KPI calculations
  • Developed DAX measures for Total Orders, Revenue, AOV, and Delivery Time
  • Implemented currency conversion for local revenue analysis
  • Analyzed trends across time, product categories, and customer segments
  • Identified patterns in online vs. in-store performance

4 Interactive Dashboard Development

  • Designed intuitive KPI card layout for key metrics
  • Created time-series visualizations for trend analysis
  • Implemented interactive filters for Store and Year dimensions
  • Applied consistent formatting and data visualization best practices
  • Optimized dashboard for executive-level consumption

Key Findings

Performance Metrics Analysis

Metric Finding Business Implication
Total Orders 20% decline since Q1 2020 Need to investigate customer acquisition and retention
Average Order Value 15% higher for in-store vs. online Opportunity to improve online upselling strategies
Delivery Time Increased by 2 days post-2020 Supply chain issues affecting customer satisfaction
Revenue by Category Computers and TVs show strongest performance Focus marketing on high-performing categories

Critical Business Insights

  • Seasonal Patterns: Q4 consistently shows 40% higher revenue than other quarters
  • Geographic Performance: European stores outperforming North American locations
  • Customer Demographics: Senior customers (60+) show highest average order value
  • Product Trends: Camera sales declining while computer sales remain strong
  • Channel Performance: Online sales growing but with lower average order value

Technical Implementation

Technologies Used

Excel PivotTables PivotCharts Slicers

Analytical Techniques

  • Data profiling and quality assessment using Power Query editor
  • Star schema data modeling with Excel's built-in Data Model
  • Advanced Excel formula implementation for calculated columns
  • Pivot Table calculated fields for business metrics
  • Time intelligence analysis using custom calendar table
  • Interactive dashboard design with slicers and timelines

Business Impact

The interactive dashboard enabled Maven Electronics leadership to:

  • Identify product categories and geographic regions driving revenue decline
  • Understand seasonal patterns to optimize inventory and marketing spend
  • Compare online vs. in-store performance to allocate resources effectively
  • Monitor delivery times and address supply chain inefficiencies
  • Segment customers by demographics for targeted marketing campaigns
Recommendation: Increase inventory and marketing for the high-performing Q4 season, implement upselling strategies to increase online AOV, replicate European success strategies in underperforming regions, consider phasing out the declining camera category, and develop loyalty programs for the high-value senior demographic. Extend the dashboard to include predictive analytics for demand forecasting and implement automated alerting for performance threshold breaches.
