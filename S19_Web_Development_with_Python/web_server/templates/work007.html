Project Overview

This project involved analyzing global video game sales data for 64,016 titles released between 1971-2024. The goal was to create an interactive dashboard that provides insights into sales trends, popular genres, top-performing consoles, and regional market differences to inform strategic decision-making for game development and marketing.

The project demonstrates my expertise in data visualization, interactive dashboard creation, and market analysis across global regions and decades of gaming history.

Business Context

Mavendo Games needed better visibility into the global gaming market to identify promising opportunities, understand regional preferences, and make data-driven decisions about game development, publishing, and market expansion strategies.

Primary Question: What are the global and regional trends in video game sales, and how can we identify the most promising markets, genres, and platforms for future game development?

Methodology & Analysis

1 Data Profiling & Quality Assurance

  • Imported and analyzed 64,016 video game titles with sales data from 1971-2024
  • Verified data types and corrected datetime formatting for release dates
  • Standardized column names for consistency (Title, Console, Genre, Publisher, Developer)
  • Created a 'release_year' feature for temporal analysis
  • Performed comprehensive data quality assessment across all regions
  • Identified and handled missing or inconsistent data points

2 Data Preparation & Aggregation

  • Created annual_sales table aggregating total sales by year
  • Generated top10_titles table ranking games by total sales
  • Prepared data for regional analysis (NA, Japan, EU, Other regions)
  • Structured data for genre-based and platform-based analysis
  • Created aggregated views for publisher and developer performance
  • Prepared datasets for interactive visualization components

3 Interactive Dashboard Development

  • Built line chart visualizing sales trends from 1971-2024
  • Created bar chart showing top 10 best-selling games of all time
  • Developed dropdown filters for Title, Genre, Publisher, Developer, and Console
  • Implemented radio button selection for sales metrics (total, NA, Japan, EU, other)
  • Designed responsive layout for optimal viewing experience
  • Ensured real-time updates across all visualization components

Business Impact

The interactive dashboard enabled Mavendo Games to:

  • Identify the most promising genres for new game development
  • Understand regional preferences for targeted marketing
  • Analyze competitor performance across different platforms
  • Identify market trends and seasonal patterns
  • Make data-driven decisions about platform focus
  • Develop region-specific game localization strategies
Recommendation: Focus development on action and sports genres for multi-platform release, with special attention to regional preferences. Consider mobile platform expansion given its growing market share, and develop region-specific marketing strategies based on the identified preferences in NA, Japanese, and European markets.
