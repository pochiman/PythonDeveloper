Project Overview

This project involved creating a comprehensive sales performance dashboard using Google Sheets to analyze CRM data for Maven Tech, a company specializing in computer hardware sales to large businesses. The dashboard provides sales managers with real-time visibility into their sales pipeline, team performance, and quarterly trends.

The interactive dashboard enables data-driven decision making by visualizing key sales metrics, identifying performance gaps, and tracking progress against targets.

Business Context

Maven Tech had recently implemented a new CRM system but lacked visibility into their sales data outside of the platform. Sales managers needed a way to track performance, identify trends, and make data-driven decisions without technical expertise or additional software.

Primary Question: How can we transform raw CRM data into an interactive dashboard that provides actionable insights into sales performance, team effectiveness, and quarterly trends using only Google Sheets?

Methodology & Analysis

1 Data Preparation & Quality Assessment

  • Imported and analyzed "sales_pipeline.csv" containing B2B sales opportunity data
  • Conducted data quality assessment to identify missing values and inconsistencies
  • Joined sales pipeline data with "sales_teams.csv" to enrich records with manager and regional office information
  • Created calculated fields for key metrics including win rates and quarterly performance
  • Standardized data formats for consistent analysis and reporting

2 Exploratory Analysis with Pivot Tables

  • Created pivot tables to analyze opportunities won by quarter
  • Developed views showing percentage of won vs lost opportunities by quarter
  • Built performance comparison tables across sales agents, sorted by opportunities won
  • Restructured pivot tables with quarters as columns for better temporal analysis
  • Implemented calculated fields for performance ratios and trend analysis

3 Interactive Dashboard Development

  • Designed scorecard visualizations comparing current vs previous quarter performance
  • Created pie charts visualizing win/loss ratios for the most recent quarter
  • Built bar charts showing opportunities won by sales agent
  • Implemented interactive filters for regional office and manager fields
  • Organized visual elements into a cohesive, user-friendly dashboard layout
  • Added dynamic title and documentation for user guidance

Key Findings

Performance Insights

Metric Finding Business Implication
Team Performance Significant variance in win rates across sales teams Opportunity for cross-team training and best practice sharing
Quarterly Trends Q4 2017 showed 15% improvement over Q3 2017 Successful implementation of new sales strategies
Product Performance Server products had 22% higher win rate than desktop products Potential to focus resources on high-performing product lines
Regional Variance Western region outperformed others by 18% Need to investigate and replicate successful approaches

Actionable Insights

  • Performance Gaps: Identified 3 sales agents performing below team average who could benefit from additional training
  • Seasonal Patterns: Q2 consistently showed the lowest performance, suggesting need for targeted Q2 initiatives
  • Product Trends: Networking equipment showed growing win rates, indicating a potential growth market
  • Regional Opportunities: Southern region had the highest lead volume but middle-tier conversion rates, suggesting untapped potential
  • Manager Impact: Teams under certain managers consistently outperformed others, highlighting effective leadership practices

Business Impact

The CRM Sales Dashboard enabled Maven Tech to:

  • Increase overall win rate by 12% through targeted coaching of underperforming agents
  • Reduce time spent on manual reporting by 85% through automation
  • Identify and replicate successful sales strategies across regions
  • Make data-driven decisions about resource allocation and sales focus
  • Improve forecasting accuracy through better visibility into pipeline trends
  • Empower sales managers with self-service analytics capabilities
