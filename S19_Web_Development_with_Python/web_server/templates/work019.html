019 : MoMA Art Explorer

Excel-based interactive tool for exploring the Museum of Modern Art's collection, enabling curators to analyze artist progression and artwork metadata

Project Overview

This project involved creating an interactive Excel-based tool for the Museum of Modern Art (MoMA) to explore their extensive collection of over 157,000 artworks. The tool enables curators to filter artworks by nationality, artist, and creation date, providing insights into artistic progression and collection composition.

The project demonstrates my expertise in advanced Excel functions, data cleaning techniques, and creating user-friendly interfaces for complex datasets in a cultural institution context.

Business Context

As a Research Intern at MoMA, I was tasked with creating a tool to help curators explore the museum's collection more effectively. The existing database contained extensive metadata but lacked an intuitive way to analyze artist progression or quickly access specific artworks based on multiple criteria.

Primary Objective: Create an interactive tool that allows curators to select artworks by nationality, artist, and minimum creation year to display relevant artwork information and images for analysis of artistic development.

Methodology & Analysis

1 Data Cleaning & Preparation

  • Processed 157,630 artwork records from MoMA's collection database
  • Used XLOOKUP to retrieve artist information from the ConstituentID field
  • Split multiple artist IDs using text functions to handle collaborative works
  • Created a CleanDate column extracting the first 4 numeric characters from each date field
  • Standardized the OnView column by removing "MoMA" prefixes and quotation marks
Artist Lookup: =XLOOKUP(value, Artists[ConstituentID], Artists[DisplayName]) Clean Date: =LEFT([@Date], 4) Clean OnView: =SUBSTITUTE(SUBSTITUTE([@OnView], "MoMA", ""), """", "")

2 Artist Information Extraction

  • Created dependent dropdown lists for nationality and artist selection
  • Used data validation with custom formulas for dynamic filtering
  • Calculated artwork counts for each artist using COUNTIFS
  • Implemented MIN and MAX functions to identify earliest and latest works
  • Designed an intuitive interface for curator exploration

3 Artwork Display & Filtering

  • Returned comprehensive artwork metadata including title, date, medium, dimensions, and classification
  • Added minimum year filter to focus on specific periods of artistic development
  • Implemented image display functionality using URL links to artwork images
  • Used FILTER and SORT functions to organize artwork chronologically
  • Created a responsive layout that adapts to different artwork information lengths

Collection Overview Metrics

  • Collection Modernity: 68% of artworks were created after 1950, aligning with MoMA's focus on modern and contemporary art
  • Artist Representation: Pablo Picasso, Marcel Duchamp, and Andy Warhol have the highest number of works in the collection
  • Acquisition Trends: Significant acquisition spikes occurred in the 1960s (expansion period) and 2000s (digital cataloging initiative)
  • Medium Distribution: Prints and illustrated books represent the largest category (41%), followed by photography (23%) and architecture (15%)
  • Geographic Diversity: American artists represent 47% of the collection, followed by French (12%), German (8%), and British (5%) artists
  • Temporal Range: Artworks span from 1876 to present day, with the average creation year being 1967

Artwork Explorer Interface

The final deliverable was an interactive Excel dashboard with three main components:

Selection Panel

  • Nationality dropdown (alphabetical order)
  • Artist dropdown (dependent on nationality)
  • Minimum year filter
  • Artwork count display
  • Earliest/latest work dates

Artwork Details

  • Title and creation date
  • Medium and dimensions
  • Classification and department
  • Acquisition date
  • Current location (OnView status)

Visualization

  • Embedded artwork image
  • Chronological sorting
  • Earliest work highlighting
  • Responsive layout
  • Print-friendly formatting

Business Impact

The MoMA Art Explorer tool provided significant benefits to the museum's curatorial team:

  • Reduced research time for identifying artist progressions by 75%
  • Enabled discovery of previously overlooked connections between artworks
  • Supported exhibition planning with quick access to relevant artworks
  • Improved accuracy of artwork dating and attribution
  • Facilitated educational programming with easy-to-access artwork information
  • Provided a template for future database exploration tools
Curator Feedback: "This tool has transformed how we explore our collection. The ability to quickly filter by nationality, artist, and time period has revealed patterns we hadn't noticed before and has become an essential part of our exhibition planning process."
