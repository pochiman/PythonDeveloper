Project Overview

This project involved creating an interactive Excel-based tool for the Museum of Modern Art (MoMA) to explore their extensive collection of over 157,000 artworks. The tool enables curators to filter artworks by nationality, artist, and creation date, providing insights into artistic progression and collection composition.

This project demonstrates advanced Excel skills in data cleaning, lookup functions, data validation, and dashboard creation for practical museum applications.

Business Context

The Museum of Modern Art needed a way for curators to efficiently explore artist progression and identify key works in their collection. With over 157,000 records spanning multiple departments and classifications, manual analysis was impractical. This tool provides an intuitive interface for curatorial research and exhibition planning.

Primary Question: How can we create an interactive tool that allows MoMA curators to easily explore artist progression and identify significant works based on nationality, artist selection, and creation date parameters?

Methodology & Analysis

1 Data Cleaning & Preparation

  • Analyzed MoMA_OnView.xlsx data across multiple worksheets containing 157,630 artwork records
  • Implemented XLOOKUP functions to retrieve artist information using ConstituentID fields
  • Developed formula to split multiple ConstituentIDs by commas and join results into single cells
  • Created a CleanDate column extracting the first 4 numeric characters from each date field
  • Standardized the OnView column by removing "MoMA" prefixes and quotation marks

2 Artist Information Extraction

  • Created dependent dropdown lists for nationality and artist selection
  • Built a second dropdown that dynamically filters Artists based on selected Nationality
  • Calculated number of artworks on display for selected artists using COUNTIFS
  • Implemented MIN and MAX functions to return the earliest and latest artwork dates
  • Designed an intuitive user interface for curator exploration

3 Artwork Display & Filtering

  • Returned comprehensive artwork metadata including Title, Medium, Dimensions, and Classification
  • Added a minimum year filter to focus on specific periods of artistic development
  • Implemented image display functionality using URL links to artwork images
  • Used FILTER and SORT functions to organize artwork chronologically
  • Implemented image display functionality using ImageURL fields within Excel cells
