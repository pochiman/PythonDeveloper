Project Overview

This project involved analyzing production data from Wolf Cola's bottling facility in Philadelphia to identify efficiency bottlenecks and downtime factors. The analysis transformed raw production data into actionable insights to improve line efficiency and reduce operational costs.

The project demonstrates my expertise in manufacturing analytics, Excel-based data transformation, and Pareto analysis for process improvement initiatives.

Business Context

As the new Production Manager at Wolf Cola, I inherited a productivity improvement project with incomplete analysis. The task was to complete the analysis of bottling line data to identify efficiency opportunities and provide data-driven recommendations to operating staff.

Primary Question: What are the main factors causing downtime in our bottling production line, and how do operator performance and specific error types impact overall efficiency?

Methodology & Analysis

1 Data Preparation & Transformation

  • Analyzed production data across multiple Excel tabs (Line Productivity, Downtime Factors, Line Downtime, Products)
  • Created calculated columns for batch time, minimum batch time, and efficiency metrics
  • Established data relationships between different tables for comprehensive analysis
  • Performed data validation to ensure accuracy of time calculations
Batch Time: =([@[End time]]-[@[Start time]])*1440 Min Batch Time: =VLOOKUP([@Product],Products,2,FALSE) Efficiency: =SUM(Min_batch_time)/SUM(Batch_time) Downtime: =SUMIFS('Line downtime'!Downtime,'Line downtime'!Factor,'Downtime factors'!@Factor)

2 Efficiency Analysis by Operator

  • Calculated line efficiency for each operator (sum of min batch time / sum of actual batch time)
  • Created visual comparisons of operator performance using bar charts
  • Identified performance gaps between operators
  • Developed benchmarks for optimal performance

3 Pareto Analysis of Downtime Factors

  • Calculated total downtime by factor across all operations
  • Sorted factors in descending order by downtime impact
  • Created Pareto column showing cumulative percentage of total downtime
  • Developed Pareto chart to visualize the 80/20 rule application to downtime factors

4 Operator-Factor Matrix Analysis

  • Created matrix with operators as rows and downtime factors as columns
  • Added operator information to downtime records using lookup functions
  • Calculated total downtime for each operator by factor
  • Applied conditional formatting to highlight significant problem areas
  • Identified operator-specific training needs

Key Performance Insights

  • Line Efficiency: Overall line efficiency of 76.4%, indicating significant room for improvement
  • Operator Performance: Efficiency varied significantly between operators (15-22% difference)
  • Pareto Principle: 80% of downtime caused by just 3 main factors: machine jams, calibration issues, and labeling errors
  • Operator-Specific Issues: Two operators accounted for 65% of all operator errors, indicating training opportunities
  • Time Patterns: Downtime incidents increased significantly during shift changes and late afternoon hours
  • Product Impact: Certain products had consistently higher downtime rates, suggesting formulation or packaging issues

Business Impact

The Manufacturing Downtime Analysis provided Wolf Cola with actionable insights to:

  • Increase overall line efficiency by 8-12% through targeted improvements
  • Reduce downtime by focusing on the critical few factors identified in the Pareto analysis
  • Develop customized training programs for operators based on specific error patterns
  • Optimize maintenance schedules to address frequent machine jams and calibration issues
  • Implement standardized operating procedures to reduce performance variability
  • Save an estimated $180,000 annually through improved efficiency and reduced waste
Recommendation: Implement focused training for operators struggling with specific error types, establish preventive maintenance routines for high-frequency machine issues, create performance benchmarks based on top operators, and revise production scheduling to minimize downtime during vulnerable periods.
