Project Overview

This project demonstrates the implementation of a robust algorithmic trading system capable of running 24/7 with comprehensive backtesting capabilities. The system integrates popular trading indicators, web scraping for market sentiment, and features a React-based dashboard for real-time monitoring.

The repository showcases my expertise in data engineering, analysis, and visualization by combining financial data processing with modern web technologies.

Key Features

  • Live Trading Bot: Connects to Oanda REST API for real-time price streaming and trade execution with multi-threading for efficient data processing
  • Backtesting System: Tests thousands of trades per second over six+ years of historical data with multiple strategy configurations
  • Technical Indicators: Implementation of MACD, RSI, and Keltner Channels for trading signals
  • Web Scraping: Collects market sentiments, headlines, and economic data to inform trading decisions
  • Interactive Dashboard: React-based frontend with Flask backend for real-time monitoring
  • Cloud Deployment: Containerized with Docker and deployed to AWS for continuous operation

Technical Implementation

The system is built with a microservices architecture, separating concerns between data collection, strategy execution, and visualization:

Data Processing

Utilizes Pandas for efficient financial data manipulation and technical indicator calculation. Historical data is stored in MongoDB for quick access during backtesting.

Strategy Engine

The core trading logic implements multiple technical indicators with parameter optimization capabilities. Each strategy can be tested against historical data to evaluate performance metrics.

Web Dashboard

The React frontend visualizes real-time market data, trading performance, and system health metrics. The Flask backend serves API endpoints for data retrieval and system control.
