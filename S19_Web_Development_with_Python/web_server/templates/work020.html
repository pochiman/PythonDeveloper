020 : CrossFit Games Analysis

Tableau-based analysis of 2019 CrossFit Games data, providing competitive insights through data profiling, transformation, and visualization

Analysis of athlete demographics including age distribution, height, and weight metrics across divisions

Country performance analysis showing medal counts, average points, and competitor distribution

Project Overview

This project involved analyzing data from the 2019 CrossFit Games to provide competitive insights for Maven CrossFit, a Boston-based gym. The analysis included profiling, cleaning, and transforming raw athlete and score data to identify patterns in athlete performance, demographic trends, and national dominance in the competition.

The project demonstrates my expertise in data preparation, transformation using Tableau, and creating actionable insights from sports performance data to drive competitive strategy.

Business Context

As a data analyst for Maven CrossFit, I was tasked with analyzing the 2019 CrossFit Games data to help the gym prepare for upcoming competitions. The raw data contained information on athletes, their scores, and demographic details but required significant cleaning and transformation to extract meaningful insights.

Primary Objective: Transform raw CrossFit Games data into actionable insights by profiling, cleaning, and analyzing athlete performance patterns to identify competitive advantages and training strategies.

Methodology & Analysis

1 Data Profiling & Quality Assessment

  • Connected to multiple data sources: athlete data (men and women), scores, and country metadata
  • Identified and removed empty rows to ensure data integrity
  • Eliminated duplicate rank fields from the scores dataset
  • Profiled initial data to understand athlete demographics and country distribution
  • Identified data quality issues and established cleaning protocols
Data Sources: 2019_games_athletes_men.xlsx 2019_games_athletes_women.xlsx 2019_games_scores.csv Country_Metadata.xlsx Quality Checks: Null value identification Duplicate field removal Data type validation

2 Data Transformation & Integration

  • Combined men's and women's athlete tables using union operation
  • Joined athlete data with country metadata using country matching
  • Integrated scores data with athlete information using left join
  • Removed redundant competitor_id field to streamline the dataset
  • Structured data for optimal analysis and visualization

3 Data Cleaning & Feature Engineering

  • Created calculated fields for height (converted to inches)
  • Engineered weight conversion to pounds for consistency
  • Developed time transformation to extract minutes as numerical values
  • Aggregated metrics by division and country for comparative analysis
  • Output cleaned datasets for further analysis in CSV and Excel formats
Height (inches): [Height] * 0.393701 Weight (pounds): [Weight] * 2.20462 Time (minutes): INT(LEFT([Time], FIND([Time], ":") - 1)) + (INT(RIGHT([Time], LEN([Time]) - FIND([Time], ":"))) / 60)

Competition Overview Metrics

  • National Dominance: United States athletes represented 47% of competitors and won 60% of medals, demonstrating home advantage
  • Peak Performance Age: Average age of top 10 finishers was 29.3 years, with peak performance between 27-32 years
  • Physical Attributes: Top performers showed optimal height-to-weight ratios rather than extreme measurements
  • Workout Specialization: Championship winners typically placed first in 3-4 individual workouts while maintaining consistency across all events
  • Gym Affiliations: CrossFit Mayhem produced the most competitors (8) and highest average finish position (12.4)
  • Gender Differences: Women competitors averaged 2.3 years older than men with similar performance patterns across age groups

Business Impact

The CrossFit Games Analysis provided significant strategic value to Maven CrossFit:

  • Informed recruitment strategy by identifying optimal athlete demographics
  • Guided training focus based on workout specialization patterns of champions
  • Developed competitive benchmarking against top-performing gyms
  • Created data-driven athlete development pathways based on age performance curves
  • Enhanced scouting operations with geographic focus on talent-rich regions
  • Established performance metrics for evaluating training program effectiveness
