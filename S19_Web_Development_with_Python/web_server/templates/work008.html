Project Overview

This project involved building regression models to predict automobile fuel economy (MPG) based on vehicle characteristics for Consumer Reports Magazine. The analysis used data from 398 cars sold in the US during the 1970s and 1980s to identify key factors influencing fuel efficiency and create accurate predictive models.

The project demonstrates my expertise in regression modeling, feature engineering, model evaluation, and interpreting results for practical applications in consumer reporting.

Business Context

Consumer Reports Magazine needed to understand the automobile characteristics that impact fuel economy for an upcoming article. Fuel economy is a major factor in the cost of car ownership, and readers rely on accurate, data-driven insights to make informed purchasing decisions.

Primary Question: Which automobile characteristics have the greatest impact on fuel economy, and how accurately can we predict MPG based on these features?

Methodology & Analysis

1 Data Preparation & Exploration

  • Loaded and analyzed automotive fuel economy data for 398 vehicles
  • Performed comprehensive data quality assessment and missing value analysis
  • Converted 'origin' to a categorical feature for proper modeling
  • Calculated summary statistics for all numeric columns
  • Created histogram of the target variable (MPG) to understand distribution
  • Built scatterplots and correlation heatmap to identify relationships
  • Identified the feature most strongly correlated with fuel economy

2 Model Development & Feature Engineering

  • Split data into training and testing sets with appropriate validation scheme
  • Built baseline regression model using the most correlated feature
  • Developed multiple regression model with feature selection
  • Performed feature engineering to improve model performance
  • Checked and addressed regression model assumptions
  • Handled multicollinearity and other potential issues
  • Implemented ridge regression for comparison (bonus challenge)

3 Model Evaluation & Interpretation

  • Evaluated model performance on test data using R² and MAE metrics
  • Interpreted model coefficients to understand feature impacts
  • Calculated the effect of model year increase on predicted mileage
  • Compared performance between multiple regression and ridge regression
  • Validated model assumptions through residual analysis
  • Prepared model for predicting fuel economy for new vehicles

Model Performance

  • Strongest Predictor: Vehicle weight showed the strongest negative correlation with fuel economy
  • Yearly Improvement: Each newer model year increased predicted fuel economy by approximately 0.8 MPG
  • Origin Impact: Country of origin significantly influenced fuel efficiency patterns
  • Engine Power: Horsepower and acceleration showed expected inverse relationships with MPG
  • Model Performance: Multiple regression achieved strong predictive accuracy (R² = 0.82)
  • Ridge Comparison: Ridge regression provided similar performance with better coefficient stability

Business Impact

The regression analysis provided Consumer Reports Magazine with:

  • Data-driven insights into factors affecting fuel economy
  • Accurate predictive model for estimating vehicle MPG
  • Understanding of how different characteristics impact fuel efficiency
  • Evidence-based content for their automotive reporting
  • Tools for comparing vehicles across different categories
  • Information to help consumers make informed purchasing decisions
Recommendation: Focus on vehicle weight and engine specifications as primary indicators of fuel economy. Consider model year improvements and country of origin patterns when comparing vehicles. Use the regression model to estimate fuel economy for new vehicle evaluations and comparisons.
