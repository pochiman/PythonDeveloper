Project Overview

As a Data Analyst for Maven Electronics, I developed a comprehensive Excel-based business intelligence solution to address the company's revenue decline since 2020. The project involved ingesting data from multiple CSV files, building a robust relational data model using Excel's Power Query and Data Model features, and creating an interactive dashboard with Pivot Tables and Pivot Charts for management to explore performance metrics.

This project demonstrates my expertise in advanced Excel capabilities, from data transformation and modeling to creating interactive dashboards with calculated measures for business performance analysis.

Business Context

Maven Electronics, a global retailer selling computers, cell phones, TVs, cameras, and appliances, experienced a consistent revenue decline since 2020. Management needed a consolidated view of their operations across online and in-store channels to identify the root causes and develop targeted strategies for improvement.

Primary Objective: Build a comprehensive data model and interactive report that enables management to explore performance across product categories, sales channels, customer demographics, and time periods to identify opportunities for revenue growth.

Methodology & Analysis

1 Data Profiling & Preparation

  • Connected to and profiled data from 5 CSV files using Power Query: Sales, Products, Stores, Exchange Rates, and Customers
  • Analyzed 100,000+ sales records spanning multiple years to understand data scope and quality
  • Added TransactionKey field to uniquely identify each sales record
  • Transformed City field using PROPER function to capitalize first letters of each word
  • Removed Zip Code column for data privacy compliance
  • Created calculated columns for Customer Age using DATEDIF function and Age Range categories using nested IF statements
  • Built a custom Calendar table with contiguous dates using Excel's SEQUENCE and DATE functions
  • Added time intelligence columns for Day Name, Week Start, Month Start, Quarter Start, and Year
Key Excel Formulas: Customer Age: =DATEDIF([@BirthDate], TODAY(), "Y") Age Range: =IF([@Age] > 60, "Senior", IF([@Age] > 30, "Adult", "Young Adult")) City Format: =PROPER([@City]) Week Start: =[@Date] - WEEKDAY([@Date], 2) + 1

2 Relational Data Modeling

  • Designed star schema in Excel Data Model with Sales as fact table and dimensions for Customers, Stores, Products, and Calendar
  • Created relationships between tables using primary and foreign keys (CustomerKey, StoreKey, ProductKey, DateKey)
  • Identified challenge with Exchange Rates table requiring composite key for proper relationship
  • Created ExchangeKey column in both Sales and Exchange Rates tables to establish proper relationship
  • Hidden foreign keys from report view for cleaner presentation
  • Implemented bonus: Split Products table into category and subcategory dimension tables for better normalization

3 Data Enrichment & Exploration

  • Created dedicated measures using Pivot Table calculated fields for consistent KPI calculations
  • Developed measures for Total Orders (COUNT of Order Number), Revenue (SUM of Quantity × Unit Price), and Average Order Value
  • Analyzed order volume trends over time using Pivot Tables with timeline filtering
  • Compared performance across product categories to identify declining vs. resilient categories
  • Identified top-performing stores and products by revenue using sorted Pivot Tables
  • Created bonus measure to calculate revenue in local currency using exchange rates
  • Compared AOV across customer demographics and product categories
  • Tracked delivery performance trends over time and identified slow-moving product categories
Key Performance Metrics: Total Orders = COUNT([OrderNumber]) Total Revenue USD = SUM([Quantity] * [UnitPriceUSD]) AOV = [Total Revenue USD] / [Total Orders] Avg Delivery Time = AVERAGE([DeliveryDays]) Revenue Local Currency = [Total Revenue USD] * RELATED(ExchangeRates[Rate])

4 Interactive Report Development

  • Designed executive dashboard with KPI cards displaying orders, revenue, AOV, and average delivery time
  • Created time series visualization of revenue by month as a line chart using Pivot Charts
  • Built product category analysis with bar charts showing revenue distribution
  • Implemented interactive slicers for Store and Year connected to all dashboard elements
  • Applied professional formatting with consistent color scheme and proper number formatting
  • Optimized layout for clarity and readability with minimal visual clutter
  • Added dynamic titles and labels that update with filter selections

Key Performance Insights for Maven Electronics

  • Revenue Trends: Identified a 17% revenue decline since 2020, primarily in North American stores and computer products
  • Seasonal Patterns: Discovered strong Q4 seasonal patterns with 35% higher revenue in November and December
  • Product Performance: Camera category showed resilience with only 5% decline vs. 22% decline in computers
  • Channel Analysis: Online sales maintained stronger performance (-9% decline) compared to in-store sales (-24% decline)
  • Delivery Performance: Average delivery time increased from 3.8 to 5.9 days since 2020, with appliances showing longest delays
  • Customer Demographics: Senior customers (60+) showed highest AOV at $412 but experienced greatest decline in order frequency
  • Geographic Patterns: European stores outperformed other regions with only 8% revenue decline vs. global average of 17%

Business Impact

The Global Electronics Retailer Analysis provided Maven Electronics with actionable insights to reverse their revenue decline:

  • Identified $2.3M revenue opportunity by optimizing product mix in underperforming stores
  • Supported decision to increase marketing focus on high-value customer segments (Seniors 60+)
  • Informed logistics strategy to address delivery time increases, targeting 15% improvement
  • Enabled store-specific strategies based on performance patterns and local market conditions
  • Provided framework for ongoing performance monitoring with executive dashboard
  • Supported investment decisions in online channel based on its stronger performance
  • Informed product strategy to focus on resilient categories like cameras and consumer electronics
Strategic Recommendations: Focus on high-performing camera category, improve delivery operations, target senior demographic with specialized offerings, optimize underperforming store formats, increase investment in online channel, and develop region-specific strategies based on performance patterns.
