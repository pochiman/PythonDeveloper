008 : Predicting Fuel Economy

Regression modeling to predict automobile fuel economy based on vehicle characteristics for Consumer Reports Magazine

Project Overview

This project involved building regression models to predict automobile fuel economy (MPG) for Consumer Reports Magazine. The analysis utilized a dataset of 398 cars from the 1970s and 1980s to identify key factors influencing fuel efficiency and develop accurate predictive models for consumer education.

The project demonstrates comprehensive regression modeling skills, from exploratory data analysis through multiple regression and ridge regression implementation, providing actionable insights for consumers and manufacturers alike.

Business Context

Consumer Reports Magazine needed to understand the automobile characteristics that impact fuel economy for an upcoming article. Fuel economy is a major factor in the total cost of car ownership, and readers rely on accurate, data-driven insights to make informed purchasing decisions.

Primary Question: Which automobile characteristics have the greatest impact on fuel economy, and how can we build an accurate predictive model to help consumers make informed decisions about vehicle purchases?

Methodology & Analysis

1 Data Preparation & Exploration

  • Loaded and examined the auto-mpg.csv dataset containing 398 vehicle records
  • Performed comprehensive data quality assessment and missing value analysis
  • Converted 'origin' to categorical feature for proper analysis
  • Calculated summary statistics for all numeric columns
  • Created histograms and distribution plots for the target variable (mpg)
  • Built scatterplots and correlation heatmaps to identify key relationships
Key Finding: Vehicle weight showed the strongest negative correlation with fuel economy (-0.83), making it the most significant predictor variable.

2 Model Development & Validation

  • Split data into training and test sets using stratified sampling
  • Implemented cross-validation scheme for robust model evaluation
  • Built baseline regression model using the strongest correlated feature
  • Developed multiple regression model with feature engineering
  • Addressed multicollinearity and other regression assumptions
  • Performed feature selection to optimize model performance

3 Model Evaluation & Interpretation

  • Evaluated model performance using R² and Mean Absolute Error (MAE) metrics
  • Interpreted coefficient values for business insights
  • Analyzed feature importance and impact on predictions
  • Validated model assumptions and residual patterns
  • Compared performance across different model configurations

4 Advanced Modeling: Ridge Regression

  • Implemented ridge regression to address potential overfitting
  • Tuned regularization parameters for optimal performance
  • Compared ridge regression results with traditional multiple regression
  • Evaluated trade-offs between bias and variance
  • Selected final model based on comprehensive evaluation metrics

Key Findings

Model Performance Results

Model Type R² Score Mean Absolute Error Key Features
Baseline (Weight-only) 0.69 3.2 MPG Vehicle weight
Multiple Regression 0.82 2.4 MPG Weight, model year, acceleration, horsepower
Ridge Regression 0.83 2.3 MPG All features with regularization

Business Insights

  • Weight Impact: For every 1,000 lb increase in vehicle weight, fuel economy decreases by approximately 6-8 MPG
  • Technological Progress: Newer model years showed significantly better fuel economy, with approximately 1 MPG improvement per year
  • Regional Differences: Vehicles from different origins (USA, Europe, Japan) showed distinct fuel economy patterns
  • Engine Power Trade-off: Higher horsepower and faster acceleration generally correlated with lower fuel efficiency
  • Consumer Guidance: The model provides reliable predictions to help consumers estimate fuel costs for different vehicle types

  • Ridge regression with L2 regularization
  • Cross-validation for model evaluation

  • Feature importance analysis
  • Model interpretation and coefficient analysis

Business Impact

The predictive models developed in this project enabled Consumer Reports Magazine to:

  • Provide data-driven fuel economy estimates for consumer vehicle comparisons
  • Identify key factors that manufacturers should focus on for efficiency improvements
  • Create educational content about the trade-offs between vehicle performance and efficiency
  • Develop interactive tools for consumers to estimate fuel costs based on vehicle characteristics
  • Make informed policy discussions about fuel efficiency standards and regulations
Recommendation: Focus on vehicle weight and engine specifications as primary indicators of fuel economy. Consider model year improvements and country of origin patterns when comparing vehicles. Use the regression model to estimate fuel economy for new vehicle evaluations and comparisons.
