Project Overview

This project involved analyzing U.S. baby name data from Social Security card applications over three decades to identify naming trends, popularity patterns, and regional variations. The analysis helped create an annual baby names report highlighting cultural shifts and emerging naming patterns.

The project demonstrates my expertise in trend analysis, demographic data interpretation, and identifying cultural patterns through data analysis.

Business Context

A popular baby names website needed to create their annual baby names report with data-driven insights about naming trends. The analysis aimed to identify how names change in popularity over time, vary across regions, and reflect broader cultural shifts.

Primary Question: How have baby naming patterns evolved over three decades, what are the regional variations in name popularity, and which names have shown the most significant changes in popularity?

Methodology & Analysis

1 Popularity Trend Analysis

  • Tracked changes in name popularity over three decades
  • Identified names with the biggest jumps and drops in popularity
  • Analyzed gender-specific naming patterns over time
  • Compared popularity rankings across different time periods
-- Names with biggest popularity jumps
WITH first_year AS (
SELECT name, COUNT(*) as count_first
FROM baby_names
WHERE year = (SELECT MIN(year) FROM baby_names)
GROUP BY name
),
last_year AS (
SELECT name, COUNT(*) as count_last
FROM baby_names
WHERE year = (SELECT MAX(year) FROM baby_names)
GROUP BY name
)
SELECT f.name, count_first, count_last,
(count_last - count_first) AS popularity_change
FROM first_year f
JOIN last_year l ON f.name = l.name
ORDER BY popularity_change DESC
LIMIT 10;

2 Decadal Comparison

  • Identified top 3 girl and boy names for each year
  • Analyzed naming trends across decades (1990s, 2000s, 2010s)
  • Compared decade-specific popularity patterns

3 Regional Analysis

  • Mapped baby name popularity across six U.S. regions
  • Identified regional naming preferences and variations
  • Analyzed geographic patterns in name popularity
  • Compared top names across different regions
-- Top names by region
SELECT region, gender, name, COUNT(*) as count
FROM baby_names
GROUP BY region, gender, name
ORDER BY region, gender, count DESC;

4 Unique Name Exploration

Key Baby Name Insights

  • Popularity Shifts: Significant changes in name popularity over time, with some names rising dramatically while others declined
  • Decadal Trends: Distinct naming patterns emerged in each decade, reflecting cultural influences and popular media
  • Regional Variations: Clear geographic patterns in name preferences, with certain names being particularly popular in specific regions
  • Gender Patterns: Traditional gender associations for names remained largely consistent, with some names becoming more androgynous over time
  • Name Length: Short names (4-5 characters) were most common, with some exceptionally long names gaining popularity in recent years
  • Cultural Influences: Popular media, celebrities, and cultural events significantly influenced naming trends

-- Androgynous names (used for both genders)
SELECT name, COUNT(DISTINCT gender) as gender_count,
SUM(CASE WHEN gender = 'F' THEN count ELSE 0 END) as female_count,
SUM(CASE WHEN gender = 'M' THEN count ELSE 0 END) as male_count
FROM baby_names
GROUP BY name
HAVING COUNT(DISTINCT gender) = 2
ORDER BY (female_count + male_count) DESC
LIMIT 10;
